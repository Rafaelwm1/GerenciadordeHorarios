📚 Gerenciamento de Horários - Escola
Projeto para gerenciar horários de professores e turmas, permitindo cadastro, filtros e exportação de dados.



🚀 Funcionalidades
🧑‍🏫 Cadastro de Professores: Adicionar informações de disponibilidade e quantidade de aulas.
🎓 Cadastro de Turmas: Registrar novas turmas.
📅 Geração de Horários: Criar horários de aula automaticamente.
🔍 Filtro de Horários: Filtrar horários por professor ou turma.
✏️ Edição de Horários: Modificar o quadro de horários existente.
📄 Exportação de Dados: Exportar horários gerados para Excel e PDF.
🛠️ Tecnologias Utilizadas
HTML5: Estruturação de conteúdo.
CSS3: Estilização da interface.
JavaScript: Lógica e interações dinâmicas.
jspdf: Geração de arquivos PDF.
xlsx: Exportação de planilhas Excel.
