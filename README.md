
📚 Gerenciamento de Horários - Escola
Projeto para gerenciar horários de professores e turmas de uma escola, permitindo cadastro, filtros e exportação de dados.

🚀 Funcionalidades
🧑‍🏫 Cadastro de Professores: Adicionar informações sobre professores.
🎓 Cadastro de Turmas: Registrar novas turmas.
📅 Geração de Horários: Criar horários de aula.
🔍 Filtro de Horários: Filtrar por professor ou turma.
✏️ Edição de Horários: Modificar o quadro de horários.
📄 Exportação de Dados: Exportar horários para Excel e PDF.
🛠️ Tecnologias
HTML5
CSS3
JavaScript
jspdf
xlsx
