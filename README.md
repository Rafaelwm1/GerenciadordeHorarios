📚 Gerenciamento de Horários - Escola
Este projeto tem como objetivo fornecer uma interface web para o gerenciamento de horários de professores e turmas em uma escola. A aplicação permite o cadastro de professores, turmas, geração de horários, filtros avançados e a possibilidade de edição de quadros de horários.

🚀 Funcionalidades
🧑‍🏫 Cadastro de Professores: Permite adicionar informações sobre os professores, como disponibilidade e quantidade de aulas por turno.
🎓 Cadastro de Turmas: Ferramenta para registrar novas turmas na escola.
📅 Geração de Horários: Gera horários de aulas com base nas informações fornecidas.
🔍 Filtro de Horários: Permite filtrar horários de professores e turmas por turno.
✏️ Edição de Horários: Permite a modificação do quadro de horários já gerado.
📄 Exportação de Dados: Exportação dos quadros de horários para Excel e PDF.
🛠️ Tecnologias Utilizadas
HTML5: Estruturação do conteúdo da página.
CSS3: Estilização da interface do usuário, incluindo responsividade.
JavaScript: Manipulação do DOM e funcionalidades interativas (filtros, geração de horários, etc.).
jspdf: Geração de arquivos PDF.
xlsx: Exportação de planilhas Excel.
